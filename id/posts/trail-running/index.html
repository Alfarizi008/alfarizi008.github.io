What is trail running?

Quote from Salomon websites:

Trail running is simply defined as going for a run in the heart of nature. It’s an extremely all-round non-stadia sport that, of course, works the legs, but also the upper body. Lastly, it requires concentration and attention.

Usually the track of trail running includes ascents and descents, rough terrain, hills, and many more, so it makes the sport more interesting yet challenging. It’s not like road running, where you always run and run; trail running usually combines running and hiking.

During the ascent part, runners often run because it makes their legs get tired too fast. Except the elite runner who has a reserve heart LOL.

In the descent part, the runners often use this to shorten or catch up on the time taken during the ascent.
